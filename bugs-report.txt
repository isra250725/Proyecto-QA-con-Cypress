ID: BUG-01
Título: Mensaje de error poco claro

Severidad: Media
Prioridad: Alta

Descripción:
El sistema no muestra un mensaje claro cuando el login falla

Pasos para reproducir:

Ir a login
Ingresar usuario válido
Ingresar contraseña incorrecta
Presionar login

Resultado esperado:
Mensaje claro indicando credenciales incorrectas

Resultado actual:
Mensaje genérico poco descriptivo
